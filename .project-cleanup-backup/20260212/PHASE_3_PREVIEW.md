# Phase 3 前端支付集成预览

## 📊 一旦准备工作完成，我们将实现：

### 🎯 核心功能

#### 1️⃣ PaymentButton 组件升级
- 调用真实支付API（WeChat Pay + Stripe）
- 二维码生成与显示（微信支付）
- Stripe Checkout 跳转
- 支付状态轮询与更新

#### 2️⃣ WeChat Pay 支付流程
```
用户点击"微信支付" → 生成订单 → 获取二维码 → 渲染二维码
    ↓
用户扫码支付 → 轮询支付状态 → 支付成功 → 更新用户层级
    ↓
关闭二维码 → 显示成功消息 → 配额立即生效
```

#### 3️⃣ Stripe Card 支付流程
```
用户点击"信用卡/支付宝" → 创建 Checkout Session → 跳转到 Stripe
    ↓
用户完成支付 → Stripe 回调 Webhook → 更新订阅状态
    ↓
返回应用 → 显示成功消息 → 配额立即生效
```

#### 4️⃣ 支付状态管理
- 实时状态跟踪（加载中/已支付/失败）
- 定时轮询支付结果
- Webhook 事件处理
- 失败重试机制

#### 5️⃣ 订阅同步
- 数据库订阅状态自动更新
- 用户层级升级（Free → Pro/Premium）
- 配额立即刷新
- 跨设备同步
